# Violence-News-Frame-Analysis
가정폭력 사건 관련 뉴스 프레임 분석을 수행한 텍스트 데이터 분석 과제

# 1. 프로젝트 개요

본 프로젝트는 2025학년도 2학기 텍스트 데이터 분석 수업에서 수행되었으며,
가정폭력 사건과 관련된 국내 뉴스 기사들이 어떠한 프레임으로 보도되는지를 분석하는 것을 목표로 한다.

데이터 수집 → 전처리 → 키워드 분석 → 프레임 분석(LDA) → 시각화의 흐름으로 구성된다.

# 2. 문제 정의 및 동기

최근 가정폭력, 아동학대, 데이트폭력, 성범죄 등 다양한 범죄가 꾸준히 발생하고 있으며,
이들 사건은 언론 보도를 통해 사회적 인식과 정책 대응에 큰 영향을 미친다.

그러나 기존 분석은 단순 키워드·통계 수준에 머물러 사건 맥락·감정·프레임 구조까지 충분히 반영하지 못하는 한계가 있다.

본 프로젝트는 이러한 한계를 극복하기 위해 뉴스 텍스트와 범죄 통계 데이터를 통합 분석하고 프레임 변화를 시계열적으로 파악하고자 한다.
