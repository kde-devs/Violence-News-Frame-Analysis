# Violence-News-Frame-Analysis
가정폭력 사건 관련 뉴스 프레임 분석을 수행한 텍스트 데이터 분석 과제

# 1. 프로젝트 개요

본 프로젝트는 2025학년도 2학기 텍스트 데이터 분석 수업에서 수행되었으며,  
가정폭력 사건과 관련된 국내 뉴스 기사들이 어떠한 프레임으로 보도되는지를 분석하는 것을 목표로 한다.

프로젝트 전체 흐름은 다음과 같다.

> 데이터 수집 → 전처리 → 키워드 분석 → 프레임 분석(LDA) → 시각화

---

# 2. 문제 정의 및 동기

가정폭력, 아동학대, 데이트폭력, 성범죄 등 다양한 범죄는 꾸준히 발생하고 있으며  
이들 사건은 언론 보도를 통해 사회적 인식과 정책 대응에 큰 영향을 미친다.

그러나 기존 분석은 다음과 같은 한계를 가진다.

> 단순 키워드·통계 중심 → 사건 맥락·감정·프레임 구조를 충분히 반영하지 못함

본 프로젝트는 이러한 한계를 극복하기 위해 뉴스 텍스트와 범죄 통계 데이터를 통합 분석하여  
프레임 변화·감정 변화·사건 서술 방식 등을 시계열적으로 파악하고자 한다.

---

# 3. 프로젝트 목표

본 프로젝트의 주요 목표는 다음과 같다.

■ 가정폭력 및 범죄 관련 뉴스 기사에서 나타나는 보도 프레임을 정량적·정성적으로 분석한다.  
■ 사건 유형별 핵심 키워드 및 서술 패턴을 추출하여 프레임 구조를 규명한다.  
■ 언론사별 보도 방식의 차이를 비교하여 편향성을 식별한다.  
■ 감성 분석 및 시계열 흐름을 통해 사회적 인식 변화를 파악한다.  
■ 뉴스 보도량과 실제 범죄 통계 간의 관계를 분석해 정책적 시사점을 도출한다.

# 7. 분석 방법

본 프로젝트의 분석 과정은 다음 세 단계로 구성된다.

## 7.1 텍스트 전처리
- 불용어·특수문자 제거  
- Okt 형태소 분석 기반 토큰화  
- 조사 및 1자 단어 제거  
- 언론사별 텍스트 구조(title, content, date) 통일  

## 7.2 키워드·감성 분석
- TF-IDF 기반 사건 유형별 핵심 단어 추출  
- 언론사별·연도별 키워드 분포 비교  
- 감정 사전 기반 감성 점수 산출  
- 불안·분노·연민 등 감성 변화 흐름 분석  

## 7.3 프레임 분석(LDA)
- CountVectorizer 기반 LDA 모델링  
- 토픽(프레임)별 핵심 단어 및 서술 패턴 도출  
- 언론사·시기별 프레임 분포 비교  
- 문서별 토픽 확률 기반 시각화(Boxplot 등)
